# SecureDataVault 🛡️

A full-stack cybersecurity application designed for secure data management, featuring a React-based dashboard and a Python-powered encryption backend.

## 🚀 Getting Started

To get this project running on your local machine, please follow the detailed step-by-step instructions in our setup guide:

👉 **[View the Full Installation & Setup Guide](./SETUP_GUIDE.md)**

## 🛠️ Tech Stack

*   **Frontend:** React, Vite, CSS3
*   **Backend:** Python (Flask/FastAPI), SQLite
*   **Security:** AES-256 Encryption, OWASP Best Practices

## ✨ Key Features

*   **Vault Dashboard:** Centralized management for sensitive notes and data.
*   **Password Strength Meter:** Real-time feedback on credential security.
*   **Trust Status & Stats:** Visual indicators of system integrity and data usage.
*   **OWASP Checklist:** Integrated security audit tools.
